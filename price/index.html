Частые вопросы

Как начать пользоваться ботом?
Запустить бота по команде /start и выбрать вариант использования.
Что входит в бесплатный пробный период?
Полный доступ ко всем функциям бота на 14 дней для одного заведения. Ограничений по функционалу нет. После окончания пробного периода необходимо выбрать один из платных тарифов.
Можно ли использовать бот для нескольких заведений?
Да, для каждого заведения необходимо создать отдельный чат в Telegram и добавить туда бота. Каждый чат будет работать с изолированной базой данных.
Как происходит оплата?
Оплата происходит напрямую в Telegram-боте через встроенную платежную систему. Оплата проходит по СБП.
Как стать партнером WAVE SELECT?
Для получения информации о программе WAVE SELECT и условиях партнерства свяжитесь с нами через Telegram @adeptlawliet. Предложение действительно только для заведений в г. Екатеринбург.
