СОГЛАСИЕ НА ОБРАБОТКУ ПЕРСОНАЛЬНЫХ ДАННЫХ

❗ Используя функционал Бота, вы подтверждаете, что ознакомились с условиями настоящего Согласия и принимаете их в полном объеме.

1. Перечень обрабатываемых данных

Оператор может обрабатывать следующие данные, которые автоматически предоставляет платформа Telegram при моем взаимодействии с Ботом:

  • Telegram User ID (уникальный идентификатор пользователя);
  • Имя и фамилия (если указаны в профиле Telegram);
  • Username (если указан в профиле Telegram);
  • Язык интерфейса Telegram;
  • Любые данные, которые используются функционалом бота

2. Цели обработки данных

Обработка моих персональных данных осуществляется в следующих целях:

  • Предоставление доступа к функциям и услугам Бота;
  • Идентификация пользователя в системе Бота;
  • Ответы на мои запросы и предоставление технической поддержки;
  • Использование функционала бота.

3. Основание для обработки данных

Основанием для обработки данных является мое добровольное согласие, выражаемое фактом начала использования Бота, а также необходимость для выполнения действий, запрашиваемых мной (предоставление услуг через Бота).

4. Способы и условия обработки

Обработка осуществляется с использованием средств автоматизации и без использования таких средств. Данные хранятся на защищенных серверах, используемых Оператором. Оператор принимает необходимые технические и организационные меры для защиты данных от неправомерного доступа.

Данные могут быть переданы третьим лицам (например, хостинг-провайдерам) исключительно для технического обеспечения работы Бота. Передача данных в страны, не обеспечивающие адекватную защиту, не осуществляется.

5. Срок действия согласия

Мое согласие действует с момента начала использования Бота до момента отзыва согласия или удаления связанных данных.
